Markdown 是什么？

Markdown 是一种轻量级的标记语言，可以把纯文本格式转换成超文本标记语言（HTML）或可扩展超文本标记语言（XHTML）文档。由于它在许多领域（如数学、图表等）都有支持，所以 Markdown 在撰写文档或发表消息中被广泛应用。

如果你是那种喜欢做笔记但不想手写的懒人，那 Markdown 十分的适合你，这是一个省时且省力的好选择。

Markdown 怎么用？

1. 如何自己使用？

注意：本安装教程针对 64 位 Windows 电脑（现在的家用电脑基本都是）。

网上可以查到相关资料，比如 Typora 就是一个很好的选择。但是 Typora 要钱（89 RMB），如果你钱不多那我还是推荐你用其他软件或者破解。破解的方法我就不详细展开了，这里讲讲 VSCode 的使用方法。

第一步：下载 VSCode

点击 该链接 下载 VSCode。看到这里的同学一定有注意中的配置，此时你可以选择链接下 Windows 下面三个选项（User Installer/System Installer/.zip）的任何一个的 x64 按钮进行下载。

如果选择了前两个选项（User Installer/System Installer）就可以打开下载的 .exe 程序按步骤安装；如果选择了最后一个选项（.zip）就把下载的 .zip 文件解压到你喜欢的位置，打开里面的 Code.exe（有需要可以创建一个快捷方式）即可。

第二步：下载相应的插件

打开 VSCode 后，进行完一系列基本设置后按下 Ctrl+Shift+X，VSCode 会自动打开 Extensions 的搜索框。搜索 Markdown All in One 和 Markdown Preview Enhanced 分别安装（点击蓝色的 Install）并等待插件和附带的插件安装完。

第三步：新建 Markdown 文件

按下 Ctrl+N 新建文件，写好后按 Ctrl+S 保存，文件的名字可以由你选择，但注意要把后缀名改成 .md（如 1.txt 要改为 1.md）。在接下来的教程中，我将教你 Markdown 的语法，也就是如何写。

第四步：查看、导出 Markdown 文件

在跟着教程学习的过程中，你看到的可能没有教程中讲到的效果。此时按下 Ctrl+Shift+V 打开预览，就可以看到效果了。如果你想一边写一边看，可以点击右上角的一个像一本书带着一个放大镜的按钮打开侧边预览。由于你安装了 Markdown Preview Enhanced 插件，预览效果可能会比 VSCode 自带的效果好。

如果你写完了，在预览的地方右键，依次选择 Export、Chrome (Puppeteer)、PDF，然后，你就会发现在同一个文件夹中多了一个同名的 .pdf 文件。如果想给其他不会用 Markdown 的人分享或转存到另一台无法打开 Markdown 文件的设备上就会很方便。

如果你对风格这方面不是很满意，可以在预览的地方右键，选择 Preview Theme，并在 Light 或 Dark 中选择自己喜欢的风格。遗憾的是，导出的 .pdf 文件似乎只有一个风格。
